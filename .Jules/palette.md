## 2024-05-16 - Using Badges for Contact Links
**Learning:** Standard text links for critical actions (like Email, LinkedIn, GitHub in a portfolio) have small hit areas, which can be difficult to tap on mobile devices or for users with motor impairments. Using Shields.io badge components provides larger, more visible click targets that improve both accessibility and visual hierarchy.
**Action:** Default to using consistent badge components (like Shields.io) or similarly styled interactive elements for important external links to improve visual UX and click target accessibility.
